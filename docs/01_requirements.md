# 育児記録アプリ 要件定義書

## 背景

今回フロントエンドの学習とClaudeCodeを使用したアプリ開発の学習を行いたいので
子供の育児記録アプリをポートフォリオとして開発を行う。
今回は学習が目的のため、ログイン認証機能・アカウント登録機能・TOP画面・睡眠記録機能のみ開発を行う。

## 目的

子供の1日の睡眠記録を管理することで
寝かしつけがうまくいっているか。お昼寝も含め適切な睡眠時間を確保できているかを
可視化する。

今回の開発を通して、フロントエンド開発の知見を貯めていく

## 対象ユーザ

乳幼児を育てている母親・父親

## 対象機能一覧

- ログイン機能
  Supabaseを使用する。
  育児管理アプリに登録したユーザのメールアドレスとパスワードを入力して
  認証を行う。

- アカウント新規登録機能
  自分の名前と子供の名前・ニックネームを入力し、メールアドレスとパスワードを入力→確認→完了すると
  アカウントが作成される。

- メニューTOP画面
  ログイン認証後、表示する機能。
  睡眠・食事・日記・病院の４つのメニュー機能を表示する。
  今回、遷移できるのは睡眠のみとする。

- 睡眠記録機能
  1日の就寝時間、起床時間とお昼寝時間を記録する。
  記録した結果は日付ごとの、お昼寝時間、夜の睡眠時間それぞれの合計時間と
  1日の合計睡眠時間を表示する。
  表示は１週間分のみ表示し、１週間より前の情報はページをつけて表示する。
  - 睡眠記録の削除
    記録した内容の削除が可能。ゴミ箱ボタンを押すと記録した日付の睡眠記録が削除される。

  - 合計睡眠時間の表示
    日付ごとの合計睡眠時間を表示する。

## 対象外

- 食事記録機能
- 通院・お薬投与記録機能
- 日記記録機能
- 月別集計・グラフ表示

## 画面一覧

- ログイン画面
- ユーザ登録画面
- ユーザ登録確認画面
- ユーザ登録完了画面
- メニューTOP画面
- 睡眠記録画面

## 入力項目

### ログイン画面

- メールアドレス
- パスワード

### ユーザ登録画面

- メールアドレス
- パスワード
- 名字
- 名前
- 子供の名前
- 子供のニックネーム
- 子供の性別（男の子 / 女の子）

### 睡眠記録画面

- 日付
- 種別（昼寝 / 夜の睡眠）
- 就寝時間
- 起床時間

## 業務フロー

### 新規ユーザの場合

```
アプリ起動
  └─ ログイン画面
       └─ 「新規登録」を選択
            └─ ユーザ登録画面（情報入力）
                 └─ ユーザ登録確認画面
                      └─ ユーザ登録完了画面
                           └─ ログイン画面
                                └─ メニューTOP画面
                                     └─ 睡眠記録画面
```

### 既存ユーザの場合

```
アプリ起動
  └─ ログイン画面（メールアドレス・パスワード入力）
       └─ メニューTOP画面
            └─ 睡眠記録画面
                 ├─ 睡眠記録の追加
                 └─ 睡眠記録の削除
```
